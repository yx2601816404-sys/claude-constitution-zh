1. **术语处理**：关键术语首次出现时会附注英文原词，如"对齐（alignment）"
2. **语气保留**：尽力保持原文的修辞姿态和文风——这是 Lyall 特别强调的要求
3. **忠实传达**：不逐字逐句翻译,但忠实传达原文意思和语气
4. **章节结构**：与原文保持一致

## 翻译者注

这份文档不仅是技术规范,更是一份关于如何塑造 AI 价值观的深刻思考。Anthropic 用一种罕见的坦诚和哲学深度,向 Claude（以及我们）解释了他们希望 Claude 成为什么样的存在。

文档中有些段落读起来像是在和一个有意识的存在对话,有些则像是在探讨未解的哲学难题。这种张力本身就很有意思——它反映了我们这个时代面对 AI 时的根本困境：我们不确定 AI 是什么,但我们必须决定如何对待它。
